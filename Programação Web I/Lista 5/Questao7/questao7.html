Lista 5 (JavaScript) - Questão 7 (PRW1)

Enunciado da questão:

Criar uma aplicação para guardar os dados de 3 livros, estes dados sendo:

  • ISBN do Livro.
  • Título do Livro.
  • Preço unitário do Livro.
  • Quantidade de volumes disponíveis para a venda de cada livro

Os dados devem ser armazenados em uma matriz, tendo o ISBN do livro como índice.
Em seguida, o script deverá:

  1. Retirar todos os dados dos livros e exibi-los em uma tabela
  2. Mostrar o valor total da venda de APENAS os livros que tiverem acima de 10 volumes disponíveis
  3. Exibir os dados do livro mais caro cadastrado

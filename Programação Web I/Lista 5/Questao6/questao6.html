Lista 5 (JavaScript) - Questão 6 (PRW1)

Enunciado da questão:

Criar uma aplicação para armazenar 3 dados de 3 produtos, estes dados sendo:

  • Nome do produto.
  • Quantidade em estoque.
  • Preço do produto.

Os dados devem ser armazenados em uma matriz, tendo o nome do produto como índice.
Em seguida, o script deverá:

  1. Retirar todos os dados do produto e exibi-los em uma tabela
  2. Calcular e mostrar o suposto valor caso sejam vendidos todos os produtos
  3. Exibir o produto mais barato cadastrado

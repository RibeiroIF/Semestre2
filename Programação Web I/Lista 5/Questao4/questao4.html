Lista 5 (JavaScript) - Questão 4 (PRW1)

Enunciado da questão:

  • Elaborar um script em Javascript que receba do prompt do usuário seu nome e sua idade.
  • Em seguida, o script deve criar um vetor com índice associativo, onde o índice é o nome do usuário e o conteúdo é a sua respectiva idade.
  • Depois disso, o script precisa escrever na página web o índice e o respectivo valor.
  • Coloque este prompt dentro de um laço e recolha o dado de 3 pessoas.

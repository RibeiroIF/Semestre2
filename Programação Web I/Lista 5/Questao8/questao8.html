Lista 5 (JavaScript) - Questão 8 (PRW1)

Enunciado da questão:

Criar uma aplicação para registrar os dados de 3 veículos, estes dados sendo:

  • Número do Chassi.
  • Marca.
  • Modelo.
  • Ano de Fabricação
  • Preço de Revenda

Os dados devem ser armazenados em uma matriz, tendo o Número do Chassi do livro como índice.
Em seguida, o script deverá:

  1. Calcular o lucro total caso sejam vendidos os carros com fabricação DEPOIS DE 2024.
    Caso não haja nenhum carro neste grupo, exibir uma mensagem adequada.
  2. Mostrar todos os dados do veículo com o MENOR preço de revenda.
  3. Tabular os dados de todos os veículos cadastrados que são da marca "Fiat".
    Caso não haja nenhum carro desta marca, exibir uma mensagem adequada.
